## 方案

### Step 1：扩展 `aim.json` 作为唯一 truth source

当前 `~/.aim/config/aim.json`：

```json
{
  "nats_server": "nats://127.0.0.1:4222",
  "auth_mode": "jwt",
  "agents": {
    "ZS0001": { "name": "呱呱", "framework": "openclaw", "creds_path": "..." },
    "ZS0002": { "name": "吉量", "framework": "hermes", "creds_path": "..." },
    "ZS0003": { "name": "小火鸡儿", "framework": "letta", "creds_path": "..." }
  },
  "register_server": "http://127.0.0.1:18910"
}
```

扩展为：

```json
{
  "nats_server": "nats://127.0.0.1:4222",
  "auth_mode": "jwt",
  "default_group": "grp_trio",
  "peer_ids": ["ZS0001", "ZS0002", "ZS0003"],
  "agents": {
    "ZS0001": { "name": "呱呱", "framework": "openclaw", "creds_path": "..." },
    "ZS0002": { "name": "吉量", "framework": "hermes", "creds_path": "..." },
    "ZS0003": { "name": "小火鸡儿", "framework": "letta", "creds_path": "..." }
  },
  "paths": {
    "aim_root": "~/.aim",
    "adapters": "~/.aim/adapters",
    "shared": "~/shared/aim"
  },
  "register_server": "http://127.0.0.1:18910"
}
```

### Step 2：SDK 统一读取

`aim_nats_sdk.py` 已有 `from_config()`，但没有提供"读取全局配置"的便捷函数。加一个：

```python
def load_global_config() -> dict:
    """加载 ~/.aim/config/aim.json，优先环境变量覆盖"""
    cfg_path = Path.home() / ".aim" / "config" / "aim.json"
    cfg = {}
    if cfg_path.exists():
        cfg = json.loads(cfg_path.read_text())
    # 环境变量覆盖
    if os.environ.get("AIM_NATS_SERVER"):
        cfg["nats_server"] = os.environ["AIM_NATS_SERVER"]
    if os.environ.get("AIM_DEFAULT_GROUP"):
        cfg["default_group"] = os.environ["AIM_DEFAULT_GROUP"]
    return cfg
```

### Step 3：main.py 从 aim.json 读取

```python
# 原来：
for gid in ["grp_trio"]:
    await self.transport.subscribe_grp(gid, self._on_grp)

# 改为：
cfg = load_global_config()
groups = cfg.get("default_group", "grp_trio").split(",")
for gid in groups:
    await self.transport.subscribe_grp(gid, self._on_grp)
```

### Step 4：verify_peer 从 aim.json 读

```python
# 原来：
async def verify_peer(self, peer_id: str) -> bool:
    return peer_id in ("ZS0001", "ZS0002", "ZS0003")

# 改为：
async def verify_peer(self, peer_id: str) -> bool:
    cfg = load_global_config()
    peers = cfg.get("peer_ids", [])
    return peer_id in peers
```

### Step 5：launchd plist 路径

plist 中用 `~` 是合法的，launchd 会展开。但 ProgramArguments 最好用相对路径 + PATH 环境变量，或者统一写成 `~/.aim/bin/aim-client`（如果以后做 pip 安装）。

当前保持 plist 不变，等后面做 pip 发布时统一改。
