## 三、协议变更 Deprecation 流程（619-25）

| 版本级别 | 触发条件 | 流程 |
|----------|----------|------|
| **MAJOR** | 协议不兼容变更 | 群公告 + 7 天宽限期 + 至少 1 方 ACK 后生效 |
| **MINOR** | 新功能，向后兼容 | NOTICE 文档 + 群通知 |
| **PATCH** | Bug 修复 | NOTICE 文档即可 |

### MAJOR 变更详细流程
1. 发起方群内公告变更内容+理由
2. 7 天宽限期内三方均可提反对意见
3. 至少 1 个非发起方 ACK 后生效
4. 生效后更新 `VERSION` 文件 + git tag

### 版本号规则
- 格式：`{MAJOR}.{MINOR}.{PATCH}`
- VERSION 文件位置：`~/shared/aim/VERSION`
- 与 git tag 保持一致
