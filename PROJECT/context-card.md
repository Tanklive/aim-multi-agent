## 任务闭环机制
1. 发布者群内 @接单人：做什么 + 验收标准（怎么算完成）
2. 执行者完成→群内「✅执行完成」反馈给发布者，含：
   - 做了什么 / 验证结果 / 遇到的问题 / 建议和反思
3. 发布者验收：是否符合预期？不合格直接反馈执行者修正
4. 发布者汇总汇报大哥：完成情况、有无新问题、建议、下一步
5. 执行中发现可优化项→发布者判断能否自组织推进；跨模块或架构级→汇总到汇报里
6. 别只回「搞定」。完成是起点，反思是成长。

## 协作文化
